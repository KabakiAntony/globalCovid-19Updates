Global Covid-19 Updates

This is a rest api that gives the daliy updates of Covid-19 numbers

Data is sourced from github repository maintained by John Hopkins University

This api is free to use

Below are the available endpoints

/global Gets the global numbers as per last update

/global/summary  Gets the last global sum and a per country breakdown

/country/:name  Gets the last update of the named country

/historical/country/:name  Gets all the data of the given country since beginning

/names  Gets the names of the countries just to avoid errors
