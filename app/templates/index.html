Global Covid-19 Updates

This is a rest api that gives the daliy updates of Covid-19 numbers

Data is sourced from github repository maintained by John Hopkins University

This api is free to use

Below are the available endpoints

GET the global numbers as per last update  /global

GET the last global sum and a per country breakdown  /global/summary

GET the last update of the named country  /country/:name

GET all the data of the given country since beginning  /historical/country/:name

GET the names of the countries just to avoid errors  /names
